doubleStruck Source #

Arguments

:: Char

The character to convert to a double struck symbol.

-> Maybe Char

The double struck symbol for the given character wrapped in a Just data constructor, Nothing if there is no equivalent double stuck character.

Obtain the double struck symbol for the given character. The supported range of characters are the alphabet characters (A–Z, and a–z), and the numerical characters (0–9). The symbols are wrapped in the Just data constructor. For characters outside the range, Nothing is returned.
