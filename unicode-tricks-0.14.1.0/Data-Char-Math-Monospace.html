monospace Source #

Arguments

:: Char

The given character to convert to a monspace symbol.

-> Maybe Char

The equivalent monospace character wrapped in a Just data constructor, Nothing if outside the alphanumerical range.

Convert the given character to its monospace equivalent for the alphabet and numerical range (A–Z, a–z, and 0–9) wrapped in a Just data constructor. For characters outside the range, Nothing is returned.
